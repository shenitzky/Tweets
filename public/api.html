Tweets API:

Method: POST /register

Parameter: userName (str), password (str), imgUrl (str)
return Boolean: True if succeed false otherwise

Method:GET /getUserStatuses

Parameter: None (user shoul be logged in)
return Json with requested statuses
Errors: If exception occured on the server

Method: POST /addUserStatus

Parameter: statusContent (str)
return Boolean: True if succeed false otherwise
Errors: If exception occured on the server

Method: POST /IncTweetForStatus

Parameter: statusId (int)
return Boolean: True if succeed false otherwise
Errors: If exception occured on the server

Method: GET /GetTop10Statuses

Parameter: None
return Json with requested statuses
Errors: If exception occured on the server

Method: GET /GetTopStatusObj

Parameter: None
return Json with requested status
Errors: If exception occured on the server

Method: GET /GetStatusById

Parameter: statusId (str)
Example: /GetStatusById?statusId=sd54654af54f64af
return Json with requested status
Errors: If exception occured on the server

Method: GET /GetAllPosts

Parameter: None
return Json with requested posts
Errors: If exception occured on the server

Method: GET /GetAllPostsSummery

Parameter: None
return Json with requested posts summery
Errors: If exception occured on the server

Method: Post /addNewPost

Parameter: postContent (str), postTitle (str), category (str)
return true id succeded false otherwise
Errors: If exception occured on the server

Method: GET /GetPostById

Parameter: postId (str)
return Json with requested post
Errors: If exception occured on the server

Method: GET /GetPostsSummeryByCategory

Parameter: category (str)
return Json with requested posts summery, limit to 3 only
Errors: If exception occured on the server

Method: GET /GetPostsByCategory

Parameter: category (str)
return Json with requested posts
Errors: If exception occured on the server

Method: GET /GetPostsCategories

None
return Json with requested post categories
Errors: If exception occured on the server
